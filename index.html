Welcome to the GitHub Pages of the Hotbit code

Hotbit is a native density-functional tight-binding calculator for python's atomic simulation environment, ASE.

Hotbit philosophy

The aim of hotbit is

  • to provide an open-source DFTB code
  • to provide a companion for DFT, for easy & fast electronic structure analysis to access to dynamical properties to investigate systems of same size for testing, playing around
  • to keep the code compact, modifiable, accessible for everyone (try to) avoid parallellization large systems size not main issue
  • to provide intuitive user interface ideal for learning and teaching realistic electronic structure simulations
